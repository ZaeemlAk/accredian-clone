# Accredian Clone

## Overview

This project is a partial clone of the Accredian Enterprise landing page built using Next.js and Tailwind CSS.

## Tech Stack

* Next.js
* React
* Tailwind CSS

## Features

* Responsive landing page
* Navbar and Footer
* Hero, Companies, Features, and CTA sections

## Setup

```bash
npm install
npm run dev
```

Open: http://localhost:3000

## Deployment

Deployed on Vercel.

## Notes

This project is created for learning and assignment purposes.
